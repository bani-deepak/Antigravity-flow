# Sanity Studio — BANI

This is the **standalone** Sanity Studio for BANI.  
Run it locally — it is NOT deployed to the public website.

## Setup

```bash
cd sanity-studio
npm install
npm run dev
```

The studio will run on [http://localhost:3333](http://localhost:3333).

## Project Details
- **Project ID**: `ywcem2gl`
- **Dataset**: `production`

## How to publish changes to the blog
1. Start the studio: `npm run dev` inside this folder.
2. Go to `http://localhost:3333`.
3. Log in with your Sanity account.
4. Create or edit posts. Published content will show up immediately on the main website.
